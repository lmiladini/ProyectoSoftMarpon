Basic list group

List groups are a flexible and powerful component for displaying a series of content.Use .list-group class with ul tag and .list-group-item with li tag.

  • 3 Facilisi natoque curabitur
  • 5 Morbi class posuere ut
  • 2 Integer laoreet odio
  • 7 Ante lobortis proin sem 
<ul class="list-group">
    <li class="list-group-item">
        <span class="tag tag-default tag-pill float-xs-right">3</span> Facilisi natoque curabitur
    </li>
    <li class="list-group-item">
        <span class="tag tag-success tag-pill float-xs-right">5</span> Morbi class posuere ut
    </li>
    <li class="list-group-item">
        <span class="tag tag-danger tag-pill float-xs-right">2</span> Integer laoreet odio
