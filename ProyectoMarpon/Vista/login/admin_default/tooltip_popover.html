Basic tooltip

Specify the tooltip positions to appear at the top, bottom, left or right.

<button type="button" class="btn btn-primary" data-toggle="tooltip" data-placement="top" title="Top Primary">
    Value...
</button>
/* Use the attribute data-toggle to specify popovers to appear from top, bottom, left or right. */

Move the mouse and notice the behaviour of the tooltip 
<div class="mouse-drag" title="Mouse follow" data-toggle="powertip" data-follow-mouse="true">

Basic popover

Use data-placement attribute to specify for the popover to appear from the top, bottom, left or right.

<button type="button" class="btn btn-primary" data-toggle="popover" data-placement="right" data-content="The quick, brown fox jumps over a lazy dog. It suddenly sleeps over it.">
    Value....
</button>
/* You can use data-placement attribute to specify for the popover to appear from the top, bottom, left or right. */

Hover popover

Use data-placement attribute to specify for the popover to appear from the top, bottom, left or right.

<button type="button" class="btn btn-primary" data-toggle="popover" data-trigger="hover" data-placement="top" data-content="hover display popover">
    Value...
</button>

/* You can use the data-placement attribute to specify for the popover to appear from the top, bottom, left or right. */
